# ✅ Что сделано

## Backend API

✅ Создан полноценный REST API для управления пентестами:
- `GET /api/pentests` - получить все пентесты
- `POST /api/pentests` - создать новый пентест
- `GET /api/pentests/:id` - получить пентест по ID
- `POST /api/pentests/:id/start` - запустить пентест
- `POST /api/pentests/:id/stop` - остановить пентест
- `DELETE /api/pentests/:id` - удалить пентест
- `GET /api/pentests/:id/logs` - получить логи пентеста

✅ Созданы модели данных:
- Pentest (пентест)
- PentestConfig (конфигурация)
- PentestLog (логи)
- Vulnerability (уязвимости)

✅ Создан сервис для управления пентестами (in-memory хранилище)

## Как использовать

1. **Откройте в браузере:** http://localhost:5173

2. **Создайте новый пентест:**
   - Нажмите "+ Новый пентест"
   - Введите название и URL цели
   - Нажмите "Создать пентест"

3. **Запустите пентест:**
   - Нажмите "Запустить" на нужном пентесте
   - Статус изменится на "Запущен"

4. **Остановите пентест:**
   - Нажмите "Остановить" на запущенном пентесте
