# Исправление Backend

## Проблема
Backend не отвечает на запросы, хотя порт 3000 слушается.

## Решение

Я создал упрощенный тестовый сервер и исправил основной сервер.

### Проверка в браузере

Откройте напрямую в браузере (не через PowerShell):
- http://127.0.0.1:3000/api/health
- http://localhost:3000/api/health

### Если не работает

1. Проверьте окно PowerShell, где запущен backend - там должны быть видны логи
2. Убедитесь, что порт 3000 свободен:
   ```powershell
   netstat -ano | findstr ':3000'
   ```
3. Попробуйте тестовый сервер:
   ```powershell
   cd C:\Xaker\backend
   npx tsx src/test-server.ts
   ```

## Изменения

- Добавлен `listen` на `0.0.0.0` для доступности на всех интерфейсах
- Создан упрощенный тестовый сервер для диагностики
- Улучшено логирование
