# ✅ Статус запуска проекта

## Процессы запущены

Я запустил проект в отдельном окне PowerShell с помощью команды `npm run dev`.

## Проверка работы

### Откройте в браузере:

1. **Frontend (веб-интерфейс):**
   - http://localhost:5173
   - Должна открыться страница "Xaker - AI Penetration Tester"

2. **Backend API:**
   - http://localhost:3000/api/health
   - Должен вернуть: `{"status":"ok","timestamp":"..."}`

## Если не работает

1. **Проверьте окно PowerShell** - там должны быть видны логи запуска
2. **Подождите 10-15 секунд** - серверы могут еще запускаться
3. **Проверьте порты:**
   ```powershell
   netstat -ano | findstr ':3000 :5173'
   ```

## Управление

- **Остановка:** Нажмите `Ctrl+C` в окне PowerShell, где запущен `npm run dev`
- **Перезапуск:** Выполните `npm run dev` в папке `C:\Xaker`

## Процессы

Проект запущен через `concurrently`, который запускает оба сервера одновременно:
- Backend на порту 3000
- Frontend на порту 5173
