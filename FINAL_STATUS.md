## Важно

Worker запускается через startup скрипт, где PATH установлен явно:
```bash
export PATH="/usr/bin:/usr/local/bin:/bin:/usr/sbin:/sbin"
```

Это означает, что когда worker процесс запускается, PATH должен быть установлен правильно.

Если ошибка `spawn node ENOENT` все еще возникает, это означает, что:
1. PATH не наследуется дочерними процессами от spawn
2. Нужно установить PATH в переменных окружения docker-compose.yml (уже сделано)
3. Или нужно патчить библиотеку @anthropic-ai/claude-agent-sdk
