# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
npm run install:all
```

## Шаг 2: Настройка backend

Создайте файл `backend/.env` на основе `backend/env.example`:

```bash
cd backend
copy env.example .env
```

Отредактируйте `.env` при необходимости.

## Шаг 3: Запуск

Из корневой папки:

```bash
npm run dev
```

Откройте браузер: http://localhost:5173

## Что дальше?

1. Изучите `PLAN.md` для понимания архитектуры
2. Изучите `SETUP.md` для детальной настройки
3. Начните разработку согласно плану
